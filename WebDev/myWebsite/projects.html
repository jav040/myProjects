Project Description
Kevin Bacon 5 degrees of freedom graph search

Created an undirected graph where a node represented an actor, and an edge was a movie connecting two nodes (actors) that were in that movie. Implemented Dijstra's graph search algorithm to find shortest path between two actors. Gained experience with class design/structure, implementing search algorithms, and I/O when reading/writing to files.
Autocomplete function

Implemented an autocomplete function, where a list of words that complete a prefix was returned to the user. I created and implemented the organization of these words using a Multiway Trie tree structure, and the words that completed a prefix were found using a depth first tree traversal search.
Markov chain text document generator

Implemented a function that reads through a directory of text files(speeches), and generated a new text file consisting of a speech that's generated based off of probabilistic likelihood of word appearances, based off of the read in words. Gained experience with C++ maps to store all the words, and I/O - reading through directories and writing to files.
Basic Haskell tokenizer and parser

Implemented an interpreter that uses Haskell to implement various language constructs. Implemented a lexer and a parser that recognized some simple constants and variables, and eventually build up to evaluating simple expressions.
This website

Finished the "Web Developer bootcamp course" on Udemy, and created this website. Gained a lot of experience with HTML, CSS, Bootsrap, JavaScript, JQuery.
